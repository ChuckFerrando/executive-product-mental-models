# Build Fast, Filter Faster Playbook

## Objective

Operate effectively in an AI-enabled environment by maximizing idea generation while maintaining strong evaluation discipline.

## Core Principle

> Generate broadly. Filter aggressively.

## Step 1 — Generate Options

- create multiple prototypes
- explore variations quickly
- avoid overthinking early

## Step 2 — Define Evaluation Criteria

Before selecting, define:

- user value
- strategic alignment
- scalability
- quality

## Step 3 — Evaluate Rapidly

For each option:

- what problem does it solve?
- does it actually work?
- what breaks at scale?
- does it align with strategy?

## Step 4 — Kill Aggressively

- eliminate weak options quickly
- aim to discard the majority
- avoid emotional attachment

## Step 5 — Double Down

- select strongest direction
- refine and iterate
- test with users

## Key Behavior Shift

Do not:
- over-plan before building

Do:
- build first, evaluate immediately

## Success Criteria

- high iteration velocity
- strong decision quality
- consistent selection of high-value ideas

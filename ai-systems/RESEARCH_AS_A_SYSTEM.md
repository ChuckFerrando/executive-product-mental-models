# Research as a System

## Definition

The shift from treating research as a one-time activity to designing it as a repeatable, automated system powered by AI.

## Core Principle

> Research is no longer something you do — it is something you run.

## Traditional Model

- Manual data collection
- Linear analysis
- One-time output
- High time cost

## System-Based Model

- Automated data gathering
- Parallel exploration
- Iterative refinement
- Continuous improvement

## System Components

1. Input
   - questions
   - hypotheses

2. Execution
   - prompts
   - agents
   - tools

3. Evaluation
   - criteria
   - scoring
   - comparisons

4. Iteration
   - refinement
   - expansion
   - optimization

## Key Insight

AI turns research into a scalable, repeatable system.

## Application

Use this model to:

- design research workflows
- improve product discovery
- run faster experiments
- build insight generation systems

- Experimentation loops
